INSTALLATION NOTES
=-=-=-=-=-=-=-=-=-=
Є Extract this package into the root of the website you wish to install soEditor
  Lite in.

Є Copy soeditor_lite.cfm into a custom tag directory. (e.g. c:\cfusion\customtags\)
 
Є Open IE and access the documentation at
  http://{www.yoursite.com}/siteobjects/soeditor/lite/docs/index.cfm

Є If you extract the package to an alternative directory, you must change 2
  ColdFusion variables in docs/index.cfm to point to the new location. in order
  for the documentation to work correctly.
